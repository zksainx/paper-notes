# 欢迎来到论文笔记

这是一个用于记录和整理论文阅读笔记的个人知识库。

## 🎯 网站简介

本站点记录了我在人工智能、机器学习等领域的论文阅读笔记。每篇笔记包含论文的核心思想、方法总结、实验结果分析以及个人思考。

## 📚 内容分类

网站按研究领域组织论文笔记：

### [Simulation](simulation/index.md)
Papers on performance modeling, simulation frameworks, and analytical models for ML systems.

### [Tensor Program Optimization](autotuning/index.md)
Papers on automated tensor program optimization through compiler techniques, ML-guided search, and performance modeling for deep learning workloads.

### [Intra-GPU Spatial Multiplexing](intra-gpu-multiplexing/index.md)
Papers on GPU spatial multiplexing techniques for efficient LLM serving, including prefill-decode disaggregation and dynamic resource partitioning.

## 🔍 如何使用

- **浏览分类**：点击顶部导航栏选择感兴趣的研究领域
- **搜索论文**：使用右上角的搜索框快速查找特定论文或关键词
- **阅读笔记**：每篇笔记包含论文摘要、核心思想、方法介绍和个人理解

## 📝 笔记模板

每篇论文笔记通常包含以下内容：

- **论文基本信息**（标题、作者、会议/期刊、年份）
- **摘要**：论文要解决的问题和主要贡献
- **核心思想**：论文的关键创新点
- **方法**：技术细节和实现方式
- **实验结果**：主要实验结论
- **个人笔记**：个人思考、疑问和延伸阅读
