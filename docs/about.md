# 关于本站

## 🎓 关于作者

我是一名对人工智能和机器学习充满热情的研究者/学习者。这个网站是我整理论文阅读笔记、沉淀知识的个人空间。

## 💡 创建初衷

在阅读大量论文的过程中，我发现：

- 论文数量多，容易遗忘之前读过的内容
- 缺少系统化的整理，难以建立知识体系
- 好的想法和启发需要记录下来

因此创建了这个网站，希望通过结构化的笔记记录：

1. **加深理解**：通过整理笔记强化对论文的理解
2. **知识沉淀**：构建个人的知识库，方便日后查阅
3. **思维碰撞**：记录阅读过程中的思考和启发

## 🛠️ 技术栈

本站点使用以下技术构建：

- **[MkDocs](https://www.mkdocs.org/)**：静态站点生成器
- **[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)**：现代化的主题
- **GitHub Pages**：免费托管
- **GitHub Actions**：自动化部署

## 📖 笔记规范

为了保持笔记的一致性和可读性，我遵循以下规范：

- **客观记录**：准确记录论文内容，不歪曲原意
- **结构清晰**：使用统一的模板组织内容
- **重点突出**：标注核心贡献和关键创新
- **个人思考**：区分论文内容和个人理解
