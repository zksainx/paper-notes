# Intra-GPU Spatial Multiplexing

Papers on GPU spatial multiplexing techniques for efficient LLM serving, including prefill-decode disaggregation and dynamic resource partitioning.

## Research Areas
- Prefill and decode phase disaggregation
- GPU SM (Streaming Multiprocessor) partitioning
- Spatial-temporal resource orchestration
- Dynamic workload scheduling
- SLO-oriented serving optimization

## Papers

### Spatial Multiplexing Frameworks
- [NanoFlow](nanoflow.md) - Optimal LLM serving with intra-device parallelism
- [Bullet](bullet.md) - Dynamic spatial-temporal orchestration for GPU utilization
- [Nexus](nexus.md) - Proactive intra-GPU disaggregation of prefill and decode
- [PD-Multiplexing](pd-multiplex.md) - SLO-oriented LLM serving with phase multiplexing
